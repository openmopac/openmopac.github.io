Notes on the file of Individual Data Sets (Home)

In general, files are sorted by empirical formula, using the order of occurrence of the atoms in the periodic table.  That is, compounds containing only carbon and hydrogen occur before compounds involving nitrogen.  Iodoform, CHI3, occurs after ferrocene, Fe(Cp)2, because iodine has a higher atomic number than iron.  In the table there are a few exceptions to this rule - these are mistakes made by the sequencing software

What it is

 This is a collection of MOPAC data sets that can be visualized using CHIME and by Jmol.  These files were used in parameterizing PM6, or were used in determining the accuracy of the various methods.

Web pages for Individual files consist of two blocks.  On the left is a picture of the data set, and on the right is a list of the data sets just before and just after this set.  To download a data set, "copy and paste" the data set below the CHIME or Jmol picture.
 
Geometry Reference data sets

 Geometry reference data are represented by data sets that include, on the third line, one or more geometric descriptors.  Thus, for ethylene, the data set would be:

 

SYMMETRY
Ethylene
GR=HERZ1966 <C-C> <C-H> <H-C-C>
C 0.00000000 0   0.0000000 0   0.0000000 0
C 1.32813567 1   0.0000000 0   0.0000000 0 1 0 0
H 1.08087917 1 123.5088506 1   0.0000000 0 2 1 0
H 1.08087917 0 123.5088506 0   0.0000000 0 1 2 3
H 1.08087917 0 123.5088506 0 180.0000000 0 1 2 3
H 1.08087917 0 123.5088506 0   0.0000000 0 2 1 5

3 1 4 5 6
3 2 4 5 6

Here, the symbol "<C-C>" means that the first geometric parameter flagged for optimization is a geometric reference, in this case the C=C distance.  The CHIME file ends in "(GEO)." Other geometric parameters are not references - they may or may not be accurate.  When it occurs, the descriptor "<>" means, "Ignore this geometric variable."
 

DHf references All other reference data are represented by CHIME and Jmol files that do NOT end in "(Geo)."  The geometries in these files are optimized using PM6, so a quick comparison of the reference geometry and optimized geometry can readily be done: most of the time, the two files are adjacent.
